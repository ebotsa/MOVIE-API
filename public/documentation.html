movie_api

This API provides a data base of movies including their title, genre, director, date of release, description, etc.

These are the endpoints of the API:

Request URL Response
GET list of all movies /movies A JSON holding all the titles as objects, e.g.:
{title: Sarafina,
director: Darrel Roodt
genre: Drama, Musical }
GET list of all users /users A JSON holding all the users as objects, e.g.:
{title: User 1,
id: 1}
GET list of all directors /directors A JSON holding all the directors as objects, e.g.:
{name: Darrell Roodt,
birthYear: 1962}
GET data about a single user by name /users/:username A JSON objects with user' data, e.g.:
{name: 'Sylvanus Agbor',
id: 1
favMovies: 'Drama, Musical',
email: 'XXXXXatyahoo.com',
password: 'mmmmm_6'
}
GET data about a movie by title /movies/:title A JSON object with title's data, e.g.:
{title: 'Sarafina',
director: Darrell Roodt
genre: Drama, Musical}
GET data about director by name /directors/:name A JSON object with directors's data, e.g.:
{name: Darrell Roodt,
birthYear: 1962}
GET data about genre by title /movies/:title A JSON object with directors's data, e.g.:
{genre: Drama, Musical }
GET list of favorite titles of a single user
by user's name		 /users/:username/favorites A JSON holding all the favorite titles of a single user, e.g.:
{title: Sarafina,
director: Darrel Roodt
genre: Drama, Musical}
